# Desafio DIO - Simulação de Ataques de Força Bruta com Kali Linux e Medusa

## 📚 Descrição do Projeto

Este projeto tem como objetivo demonstrar a utilização do Kali Linux e da ferramenta Medusa em um ambiente controlado para simulação de ataques de força bruta em diferentes serviços vulneráveis.

Foram utilizados ambientes vulneráveis como o Metasploitable 2 e o DVWA (Damn Vulnerable Web Application) para fins exclusivamente educacionais e laboratoriais.

---

# ⚠️ Aviso Importante

Todos os testes realizados neste projeto ocorreram em ambiente controlado e autorizado, com foco educacional e de aprendizado em segurança ofensiva e defensiva.

Nenhum teste foi realizado contra sistemas reais ou sem autorização.

---
